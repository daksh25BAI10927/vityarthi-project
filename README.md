# Python Expense Tracker (OOP)

A simple console-based Expense Tracker implemented in Python (OOP).  
This project stores expenses, allows viewing and sorting, and persists data to a JSON file.

## Features
- Add expense with automatic date (today)
- Add expense with custom date (YYYY-MM-DD)
- View all expenses
- Sort by amount (asc / desc)
- Group expenses by date (headline per date)
- Data persistence using `expenses.json`
- Basic input validation and error handling
